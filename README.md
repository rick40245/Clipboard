# Clipboard

Clipboard 是一個易於使用的 Windows 應用程式，它可以快速地複製和貼上文字，並允許使用者管理多個複製項目。

## 特點

- 管理多個複製項目
- 快速複製和貼上功能
- 將複製項目保存到磁盤以便於重啟後使用

## 安裝

從發布頁面下載最新版本的 .exe 檔案，然後雙擊該檔案進行使用。

## 使用

啟動應用程式後，你會看到一個有多個文字欄位和按鈕的介面。你可以在每個文字欄位中輸入你想要保存的文字，然後點擊對應的 "複製" 按鈕，該文字就會被貼上到當前焦點位置。你也可以點擊 "刪除" 按鈕來移除一個複製項目。

要新增一個新的複製項目，你可以點擊 "新增" 按鈕。要保存所有的複製項目，你可以點擊 "保存" 按鈕，這將把所有的複製項目保存到磁盤，下次啟動應用程式時，它們將自動加載。
